# AI Website Manager

This AI agent automatically optimizes a WordPress.com website and custom HTML files using the OpenAI API.

## Features
- WordPress SEO optimization
- WordPress content rewriting
- Custom HTML optimization via FTP
- Daily automatic cron tasks
- Runs on Render.com

## Files
- agent.js: Main controller
- wordpress.js: WordPress automation
- htmlManager.js: HTML automation
- cron.js: Scheduled tasks
